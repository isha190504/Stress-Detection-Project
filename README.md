# Stress Detection Project

This project predicts human stress levels using physiological sensor data and machine learning.

## Technologies Used
- Python
- TensorFlow
- Random Forest
- Signal Processing

## Features
- Real-time sensor data collection
- CNN feature extraction
- Random Forest classification
- Stress prediction with confidence score

## Files
- train.py → CNN + RF training pipeline
- main.py → Real-time stress prediction
- cnn_feat_mod.h5 → CNN feature extractor
- rf_mod.pkl → Random Forest classifier

## Run Project

```bash
python main.py
```
